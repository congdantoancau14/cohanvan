## 📖 Introduction

**Cohanvan** (Cổ Hán Văn) is a dedicated project focused on the preservation, study, and digital processing of Classical Chinese (Hán văn), Hán-Việt phonology, and Chữ Nôm. 

As digital landscapes change, valuable linguistic tools and data can easily disappear from the live web. This repository serves as a restored archive and a continuing foundation for tools that bridge historical ideographic scripts with modern Vietnamese digital processing.

---

## ✨ Key Features & Scope (Future)

*   **Linguistic Mapping:** Tools and datasets for mapping Classical Chinese characters to their correct Hán-Việt readings.
*   **Historical Etymology:** Resources dedicated to understanding the structural evolution of Chữ Nôm and early Vietnamese scripts.
*   **Digital Preservation:** A reliable, open-source reference for developers, linguists, and hobbyists working on Sino-Vietnamese digital humanities.
